Proxmox Setup

Virtualisation

Proxmox is hosted on my Dell R710 server. With 24 threads, 32GB of RAM and 2 TB of storage. Whilst the R710 is a powerfull system, it is very power hungry. With a minimum power draw of 180w. As a result I have made the decision to only run the server when I wish to mess around with virtual machines. It’s main purpose is for developing more solid applications that can be deployed to more power efficient systems.

Current VM’s in use

  • pfSense
  • Ubuntu Server (Kuberntes nodes at the current time)
  • Windows 10
  • Windows Server 2019

Modifications

I have a script that runs constantly from a Rapberry Pi that manually reduces the power of the servers fans. This means that the server is able to be left on without anyone complaining about noise (It lives in my room). With the software hack the sound is about the same sound as a typical desktop PC.

The one downside to this script though is I have to adjust the fan speed based upon the ambient temperature and not on the CPU temperature. According to the homelab community Dell just seem to have removed the ability to see the CPU temperature. as it isn’t visible withing the Dell iDrac interface or bios either.

Potential upgrades

  • Install an IT flashed HBA card to enable ZFS to be used on the installed drives.
  • Install SSD’s to reduce power consumption.
  • (Bit extreme) Run the server off of a solar panel and battery bank.
